# Budong Art Center

不东艺术中心网站项目。

## Structure

- `index.html` - GitHub Pages entry page.
- `styles.css`, `script.js`, `smokey-cursor.js` - main site styles and interactions.
- `assets/pdf/` - images extracted and curated from the source proposal PDF.
- `assets/generated/` - generated visual assets used during design.
- `demos/incense/` - incense smoke interaction prototype.
- `docs/source/` - original source material.
- `docs/verification/` - local visual verification screenshots.
- `docs/contact-sheets/` - image contact sheets.

## Publish

This repo is organized so GitHub Pages can serve the site directly from the repository root.
